# phylum ping

Ping the remote system to verify it is available

```sh
Usage: phylum ping [OPTIONS]
```

## Options

`-v`, `--verbose`...
&emsp; Increase the level of verbosity (the maximum is -vvv)

`-q`, `--quiet`...
&emsp; Reduce the level of verbosity (the maximum is -qq)

`-h`, `--help`
&emsp; Print help
