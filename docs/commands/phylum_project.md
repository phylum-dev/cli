# phylum project

Manage Phylum projects

```sh
Usage: phylum project [OPTIONS] <COMMAND>
```

## Options

`-v`, `--verbose`...
&emsp; Increase the level of verbosity (the maximum is -vvv)

`-q`, `--quiet`...
&emsp; Reduce the level of verbosity (the maximum is -qq)

`-h`, `--help`
&emsp; Print help
