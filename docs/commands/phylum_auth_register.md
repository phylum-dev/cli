---
title: phylum auth register
---

Register a new account

```sh
Usage: phylum auth register [OPTIONS]
```

### Options

`-n`, `--token-name`
&emsp; API token name

`-v`, `--verbose...`
&emsp; Increase the level of verbosity (the maximum is -vvv)

`-q`, `--quiet`...
&emsp; Reduce the level of verbosity (the maximum is -qq)

`-h`, `--help`
&emsp; Print help
