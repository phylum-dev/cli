# phylum auth status

Return the current authentication status

```sh
Usage: phylum auth status [OPTIONS]
```

## Options

`-v`, `--verbose`...
&emsp; Increase the level of verbosity (the maximum is -vvv)

`-q`, `--quiet`...
&emsp; Reduce the level of verbosity (the maximum is -qq)

`-h`, `--help`
&emsp; Print help
