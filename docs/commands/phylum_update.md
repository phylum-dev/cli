---
title: phylum update
---

Update to the latest release of the Phylum CLI

```sh
Usage: phylum update [OPTIONS]
```

### Options

`-v`, `--verbose`...
&emsp; Increase the level of verbosity (the maximum is -vvv)

`-q`, `--quiet`...
&emsp; Reduce the level of verbosity (the maximum is -qq)

`-h`, `--help`
&emsp; Print help
