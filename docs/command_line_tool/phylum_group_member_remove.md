Remove user from group

```sh
phylum group member --group <GROUP> remove [OPTIONS] <USER>...
```

### Arguments

`<USER>`...
&emsp; User(s) to be removed

### Examples

```sh
# Remove user `demo@phylum.io` from the `sample` group
$ phylum group member --group remove demo@phylum.io
```
