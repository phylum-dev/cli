List group members

```sh
phylum group member --group <GROUP> list [OPTIONS]
```

### Options

`-j`, `--json`
&emsp; Produce member list in json format (default: false)

### Examples

```sh
# List all group members for the 'sample' group
$ phylum group member --group sample list
```
