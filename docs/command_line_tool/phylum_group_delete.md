Delete a group

```sh
Usage: phylum group delete [OPTIONS] <group_name>
```

### Arguments

<group_name>
&emsp; Name for the group to be deleted

### Options

-v, --verbose...
&emsp; Increase the level of verbosity (the maximum is -vvv)

-q, --quiet...
&emsp; Reduce the level of verbosity (the maximum is -qq)

-h, --help
&emsp; Print help
