Add user to group

```sh
phylum group member --group <GROUP> add [OPTIONS] <USER>...
```

### Arguments

`<USER>`...
&emsp; User(s) to be added

### Examples

```sh
# Add user `demo@phylum.io` to the `sample` group
$ phylum group member --group add demo@phylum.io
```
