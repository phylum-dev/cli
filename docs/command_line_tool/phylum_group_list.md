List all groups the user is a member of
```sh
phylum group list [OPTIONS]
```

### Options
`-j`, `--json`
&emsp; Produce output in json format (default: false)


### Examples
```sh
# List all groups the user is a member of
$ phylum group list

# List all groups the user is a member of with json output
$ phylum group list --json
```
