```sh
phylum project list [OPTIONS]
```
List all existing projects.

### Options
`-j`, `--json`
&emsp; Produce output in json format (default: false)

### Examples
```sh
# List all existing projects
$ phylum project list

# List all existing projects with json output
$ phylum project list --json
```
